title: 'RMA Event'
description: 'Simulate RMA receiving and processing events (Sandbox only)'
api: 'POST /data_factory/rmaEvent'
---

Simulate RMA (Return Merchandise Authorization) receiving and processing events to test return workflows.

<Warning>
DataFactory is only available in the sandbox environment.
</Warning>

## Request Body

<ParamField body="customer_identifier" type="string" required>
  RMA customer identifier.
</ParamField>

<ParamField body="event" type="string" required>
  RMA event type. Options: `receiveFull`, `receivePartial`, `tendered`, `inTransit`, `delivered`.
</ParamField>

<ParamField body="randomize_inventory_statuses" type="boolean" default="false">
  Randomize return inventory statuses for testing quality scenarios.
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://app.masonhub.co/{account}/api/v1/data_factory/rmaEvent" \
  -H "Authorization: Bearer your_jwt_token" \
  -H "Content-Type: application/json" \
  -d '[{
    "customer_identifier": "rma123",
    "event": "receiveFull",
    "randomize_inventory_statuses": true
  }]'
```

```python Python
import requests

url = "https://app.masonhub.co/{account}/api/v1/data_factory/rmaEvent"
response = requests.post(url,
    headers={"Authorization": "Bearer your_jwt_token"},
    json=[{
        "customer_identifier": "rma123",
        "event": "receiveFull",
        "randomize_inventory_statuses": True
    }])
```

```javascript JavaScript
await fetch('https://app.masonhub.co/{account}/api/v1/data_factory/rmaEvent', {

  body: JSON.stringify([{

```
</RequestExample>

## Available Events

<AccordionGroup>
  <Accordion title="Receive Full">
    Receive all expected items from the RMA
  </Accordion>
  <Accordion title="Receive Partial">
    Receive random subset of expected items
  </Accordion>
  <Accordion title="Tendered">
    Return package tendered to carrier
  </Accordion>
  <Accordion title="In Transit">
    Return package in transit to facility
  </Accordion>
  <Accordion title="Delivered">
    Return package delivered to facility
  </Accordion>
</AccordionGroup>

## Randomization

When `randomize_inventory_statuses` is true, returned items are received into random statuses:
- available (resaleable)
- damaged
- quality-control
- refurbishing
- under-investigation
